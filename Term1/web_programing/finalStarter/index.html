Simple Interest Calculator

Instructions:

  • Retrieve Principal, Interest Rate and Duration From the UI
  • Calculate Interest with the following formula: Interest = Principal * (Interest Rate/ 100) * Duration
  • Display Interest Back to User (rounded to 2 Decimals) with an appropriate message

Fuel Consumption Rate Calculator

Instructions:

  • Retrieve Distance Travelled and Liters of Fuel Used from the UI
  • Calculate Consumption Rate = Distance / Liters Used
  • Display Consumption Rate back to User (rounded to 2 Decimals) with an appropriate message

Distance Conversion

Instructions:

  • Retrieve Distance from the UI
  • Convert to Miles (Distance * 0.62137)
  • Convert to Kilometers (Distance * 1.60934)
  • Display Converted Distance Back to User (rounded to 2 Decimals) with an appropriate message

Load an Array of Grades

Instructions:

  • Retrieve a Series of Grades from the UI
  • Store each Grade Retrieved in an Array
  • Display the Array Back to the User Each Time an Grade is Entered
  • Give the User the ability to Reset the Array

Calculate the Average of all Grades in the Above Array

Instructions:

  • There is no User Input from the UI other than Clicking the Calculate Average Button
  • Use an Algorithm to Average Grade (Total Grade / Number of Grades Entered) from the Previous Array
  • Display the Average Grade back to the User with an Appropriate Message

Create a Simple Array of Objects

Instructions:

  • Retrieve the Information from the UI
  • Store the Information into an Object
  • Store the Object in an Array

Character Scroller

Instructions:

  • There is no User Input from the UI other than Clicking the Load, Forward and Back Buttons
  • Create the Load, Forward and Back Buttons
  • Load button should Load a Random Character
  • Forward and Back Buttons should Loop Indefinitely
  • Use the Array of Objects built Above and Display each Character back to the User on Demand
