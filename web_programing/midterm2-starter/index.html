Midterm 2 - Practical Exam

Eleminate Redundancy at every Opportunity

Tip Calculator (10 Marks)

Instructions:

  • Retrieve Bill Amount and Tip Percent From UI
  • Calculate Tip
  • Display Tip Back to User (rounded to 2 Decimals)

Bill Splitter (15 Marks)

Instructions:

  • Retrieve Bill Amount, Tip Percentage and Number of People from UI
  • Calculate Tip, Total and split it
  • Display Split Amount back to User (rounded to 2 Decimals)

Temperature Conversion (10 Marks)

Instructions:

  • Retrieve Temperature from the UI
  • Convert Celsius to Fahrenheit ((temp C * 9/5) + 32)
  • Convert Fahrenheit to Celsius ((temp C -32) * 5/9)
  • Display Converted Temperature Back to User (rounded to 2 Decimals)

Load an Array of Cars (15 Marks)

Instructions:

  • Retrieve a Series of Input from the UI
  • Store each Item Retrieved in an Array (make sure car is not already in array)
  • Display the Array Back to the User Each Time an Item is Entered
  • Give the User the ability to Reset the Array

Bonus: Pick a Random Car from the Array every time the User Clicks the button (+3 Marks)

Instructions:

  • There is no User Input from UI other than Clicking the Button
  • Use an Algorithm to Select One of Cars in the Previous Array
  • Display the Car back to the User with Appropriate Feedback
